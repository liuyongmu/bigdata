# Hive

##### 数据类型

~~~mathematica
数据类型    描述
BOOLEAN	   true/false   
TINYINT	   1字节的有符号整数，表示范围: -128~127
SMALLINT   2个字节的有符号整数，表示范围: -32768~32767
INT	       4个字节的带符号整数，表示范围: -2147483648~2147483647
BIGINT	   8字节带符号整数，表示范围: -2^64~2^64-1
FLOAT	   4字节单精度浮点数
DOUBLE	   8字节双精度浮点数
DEICIMAL   任意精度的带符号小数
STRING	   字符串
VARCHAR	   变长字符串
CHAR	   固定长度字符串
BINARY	   字节数组
TIMESTAMP  时间戳
DATE	   日期
ARRAY	   数组类型
MAP	       key-value,key必须为原始类型，value可以任意类型	map(‘a’,1,’b’,2)
STRUCT	   段集合,类型可以不同
~~~
